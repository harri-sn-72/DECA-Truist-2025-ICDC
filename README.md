# DECA-Truist-2025-ICDC
Developed this app using Swift and Xcode in early 2025. Prototype to present at both North Carolina DECA CDC and ICDC. Used CoreData ML, Google Gemini API, and integration with native Apple devices. This project was part of a larger business proposal for artificial intelligence integration into Truist Bank that was presented to judges at the state and international levels.
